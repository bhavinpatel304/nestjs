# nestjs

Repo contains two folders backend and frontend

:: In backend :: 
1) npm install
2) npm start
    <br/>
    Backend must run on port 3000.
    <br/>
    Url setting :: http://localhost:3000/
3) Now run test if you want
    use : npm test
